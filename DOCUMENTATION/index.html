A) HTML Structure - top

This template is a responsive multipurpose fixed layout with one column (in some sections are few columns). All sections of this template are nested in div with class layout. Every section has own #id (name). Here is the general structure:

  1. header
  2. #subheader
  3. #about section
  4. #filter section
  5. #heading-slogan section
  6. #video section
  7. #pricing section
  8. #customers section
  9. #follow section
  10. #team section
  11. #blog section
  12. #contact section
  13. #map section
  14. #theme section
  15. #social section
  16. footer

Whole code in HTML file are well commented, and indented.

If you would like to edit the color, font, or style of any elements in one of these columns, you can do that in CSS files located inside css folder.

B) CSS and SCSS Files and Structure - top

Our theme using one CSS file located in css folder. It is style.css file.

The file is separated into sections using:

	
		/* --- +  BASIC STYLES + --- */

		/* --- +  ANIMATIONS + --- */

		/* --- +  LAYOUT + --- */
		
		/* --- +  HEADER + --- */
		
		/* --- +  BUTTONS + --- */
		
		/* --- +  FORMS + --- */
		
		/* --- +  HEADER + --- */
		
		/* --- +  VALIDATION POPUP + --- */

		/* --- +  SCROLL TO TOP + --- */

		/* --- +  CONTENT + --- */

		/* --- +  MEDIA --- */

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

C) JavaScript - top

This theme imports several Javascript files located in js folder.

  1. jQuery - JavaScript Library
  2. Map Api
  3. Selectivizr - JavaScript utility that emulates CSS3 pseudo-classes and attribute selectors in Internet Explorer 6-8.
  4. Modernizr - JavaScript library that detects HTML5 and CSS3 features in the user’s browser.
  5. jQuery Validation - Form validation with jQuery. Used in contact form.
  6. Izotope jQuery - An exquisite jQuery plugin for magical layouts - Used in #filter section for filtering filter elements.
  7. Respond.js - A fast & lightweight polyfill for min/max-width CSS3 Media Queries (for IE 6-8, and more)
  8. Magnific Popup is used in #filter section.js
  9. Animate Number is used in #blog section under filter elements
  10. Inview plugin is used for element with animate when thay visible
  11. mb.YTPlayer
  12. Masonry sorting elements
  13. OWL slider.js
  14. Our custom JS scripts - init.js
